## Introduction

Sluggable is a lightweight and flexible Laravel package that provides automatic, customizable slug generation for your Eloquent models.
Using the HasSlug trait, your models gain the ability to generate SEO-friendly slugs from one or more fields, prevent duplicates, customize separators, control update behavior, and more.
